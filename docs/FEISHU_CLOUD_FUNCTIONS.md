# 飞书云函数迁移设计

## 订单同步函数

定时拉取抖音小店、天猫、拼多多订单，做字段清洗后写入飞书多维表格或 ERP 订单池。

## 库存预警函数

定时读取 WMS 库存，发现低库存 SKU 后创建异常记录和采购补货审批。

## 物流异常函数

定时查询物流轨迹，识别揽收延迟、运输停滞、退回、少发漏发风险，并推送客服跟进。

## AI日报定时函数

每天 18:00 汇总订单、库存、异常、物流、财务和客户线索，调用多模型路由生成日报，再推送飞书群。

## Webhook 通知函数

封装飞书机器人 Webhook，统一处理日报推送、P1/P0 异常提醒、审批结果通知和任务超时提醒。

## 迁移原则

本地 FastAPI 适合面试演示和开发调试；飞书云函数适合把同步、预警、日报等定时任务迁移到低代码平台内运行。
