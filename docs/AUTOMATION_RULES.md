# 自动化规则设计

## 1. 订单异常触发规则

### 规则名称
订单状态变更为异常 → 推送飞书/企业微信群

### 触发条件
订单状态 = 异常

### 触发动作
- 创建异常记录
- 推送消息到群机器人
- 生成负责人待办

### 消息模板

```text
【订单异常提醒】
订单号：{order_no}
客户：{customer}
异常类型：{exception_type}
原因：{reason}
负责人：{owner}
优先级：{priority}
请在 {due_date} 前处理。
```

## 2. 库存预警规则

### 触发条件
当前库存 < 安全库存

### 触发动作
- 将 SKU 加入库存预警列表
- 推送补货提醒
- 创建采购/供应商确认待办

### 消息模板

```text
【库存预警】
SKU：{sku}
商品：{product_name}
当前库存：{current_stock}
安全库存：{safety_stock}
库存缺口：{gap}
供应商：{supplier}
```

## 3. 交期风险规则

### 触发条件
交期临近 AND 订单未完成

### 触发动作
- 标记为交期风险
- 推送负责人
- 加入日报重点风险

## 4. 客户线索沉淀规则

### 触发条件
AI客服识别到报价、客服、网站、机器人、自动化等关键词

### 触发动作
- 创建客户线索
- 判断意向等级
- 推送销售/负责人
- 加入日报客户线索板块

## 5. 每日业务日报规则

### 触发时间
每天固定时间，例如 18:00 或 20:00

### 输入数据
- 今日订单总数
- 未关闭异常
- 低库存 SKU
- 未完成待办
- 新客户线索

### 输出内容
- 今日总览
- 重点风险
- 库存预警
- 优先处理事项
- 客户线索
- AI处理建议
